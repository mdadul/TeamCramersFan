Inverse Matrix

The Inverse Matrix method involves finding the inverse of a square matrix to solve a system of linear equations.

Consider a system of linear equations:

                
                    Ax = B

Where A is the coefficient matrix, x is the vector of variables, and B is the constant matrix.

To solve for x, we find the inverse of A and multiply both sides by it: 

                
                    x = A-1 * B

Let's consider an example:

                
                    A = | 2  1 |
                        | 4 -3 |

                    B = | 5 |
                        |-3 |

The inverse of A is:

                
                    A-1 = | 3  1 |
                          | 4  2 |

The solution x is then:

                
                    x = A-1 * B
                      = | 3  1 |   | 5 |   |  8 |
                        | 4  2 |   |-3 |   |  6 |

Cramer's Rule

Cramer's Rule provides a method to solve a system of linear equations by expressing the solution in terms of determinants.

Consider the system of linear equations:

                
                    Ax = B

The solution for variable xi is given by: 

                
                    xi = det(Ai) / det(A)

Where Ai is the matrix obtained by replacing the i-th column of A with the column vector B.

Let's consider an example:

                
                    A = | 2  1 |
                        | 4 -3 |
    
                    B = | 5 |
                        |-3 |

The determinants are:

                
                    det(A) = -10
    
                    det(A1) = |  5  1 |
                              | -3 -3 |
                            = 16
    
                    det(A2) = |  2  5 |
                              |  4 -3 |
                            = 26

The solutions are then:

                
                    x1 = det(A1) / det(A) = -1.6
                    x2 = det(A2) / det(A) = -2.6

Gauss Elimination

Gauss Elimination is a systematic method for solving systems of linear equations by transforming the augmented matrix to its row-echelon form.

Consider the augmented matrix [A | B]:

                
                    | 2  1 | 5 |
                    | 4 -3 |-3 |

Perform row operations to transform it to row-echelon form: 

                
                    | 1  0 |  3 |
                    | 0  1 | -2 |

The system of equations is now simplified, and the solutions can be directly read off.

The solution is:

                
                    x1 = 3
                    x2 = -2

Jacobi Iterative Method

The Jacobi Iterative Method is an iterative approach to solving systems of linear equations by updating the values of the variables in each iteration.

Consider the system of linear equations:

                
                    2x1 + x2 = 5
                    4x1 - 3x2 = -3

Initialize the variables and update them iteratively:

                
                    Iteration 1:  x1 = 0, x2 = 0
                    Iteration 2:  x1 = 2.5, x2 = -1.25
                    Iteration 3:  x1 = 1.625, x2 = -0.375
                    ...

Continue iterating until convergence is reached, i.e., the values stabilize.
